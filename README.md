# Gsap-website
# 🏌️ Sidcup Family Golf Website Clone

A modern, animated golf website built using **HTML, CSS, JavaScript and GSAP**.  
This project focuses on smooth animations, interactive UI, and modern web design.

---

## 🚀 Features

- 🎥 Background video hero section
- 🎯 GSAP animations (scroll + hover effects)
- 🖱️ Custom cursor animation
- 🔄 Infinite scrolling text effect
- 🎨 Modern UI design
- 📱 Responsive layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GSAP (GreenSock Animation Platform)

---

## 📂 Project Structure

```
gsap-website/
│── index.html
│── style.css
│── script.js
│── README.md
```

---

## ▶️ How to Run

1. Clone the repository
```
git clone https://github.com/Asma449/gsap-website.git
```

2. Open project folder

3. Run using Live Server OR open `index.html` in browser
